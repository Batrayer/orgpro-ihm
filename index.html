Commande Description
task collaborator/col add "numéro de la tâche" "nom du collaborateur" Associer un collaborateur à une tâche.
task collaborator/col delete "numéro de la tâche" "nom du collaborateur" Supprimer un collaborateur d'une tâche.
task clock use "numéro de la tâche" Lancer/stopper le chronomètre d'une tâche.
task state "numéro de la tâche" "nom de l'état" Changer l'état d'une tâche.
task property/prop add "numéro de la tâche" "nom de la propriété" Ajouter une propriété à une tâche.
task property/prop delete "numéro de la tâche" "nom de la propriété" Supprimer une propriété d'une tâche.
task tag add "numéro de la tâche" "nom du tag" Ajouter un mot-clef à une tâche.
task tag delete "numéro de la tâche" "nom du tag" Supprimer un mot-clef d'une tâche.
task deadline/dl "numéro de la tâche" "date limite" Ajouter/modifier la date limite d'une tâche.
task closed/cl "numéro de la tâche" "date d'accomplissement" Ajouter/modifier la date d'accomplissement d'une tâche
task scheduled/sd "numéro de la tâche" "date de commencement" Ajouter/modifier la date de commencement d'une tâche
task rename "numéro de la tâche" "nom de la tâche" Renommer le titre d'une tâche.
task add "nom de la tâche" ("numéro de la tâche à dépendre") Ajouter une nouvelle tâche. (Elle peut-être dépendante d'une tâche)
task delete "numéro de la tâche" Supprimer une tâche.
task list Lister les tâches avec leur numéro.
task dep set "numéro de la tâche" "numéro de la tâche à dépendre" Ajouter/modifier la dépendance d'une tâche.
task dep delete "numéro de la tâche" Supprimer la dépendance d'une tâche.
task help Afficher l'aide de la commande "task".
file list Afficher la liste des fichiers ".org".
file select Selectionner un fichier de travail.
file help Afficher l'aide de la commande "file".
list task Afficher le contenu du fichier.
list ts "nom de l'état" Lister les tâches ayant l'état indiqué.
list ats Lister toutes les tâches.
list sd Lister les tâches dont la date de commencement est déjà passée et ayant l'état TODO.
list help Afficher l'aide de la commande "list".
